# Jarvis_FaceNet

A professional desktop voice assistant that uses FaceNet (512-D embeddings) trained on your own images. Features:
- Wake word detection ("Hey Jarvis")
- Face recognition (FaceNet embeddings, your dataset)
- Create files & folders by voice
- System control: shutdown, restart, lock, volume
- WhatsApp sending (pywhatkit)
- Alarm system
- Notes / to-dos (persistent JSON)
- Jokes / facts / quotes
- Dynamic app launcher (data/apps.json)
- Screenshot capture
- Read news (NewsAPI)
- Get weather (OpenWeatherMap)
